Hotkeys

Summarization of hotkeys for Blender's tools.

View

  • Mouse-middle-click(drag): Rotate view
  • Shift + Mouse-middle-click(drag): Translate view
  • [Select object]' . ' from numberpad: Focus view on object
  • '0' from numberpad: Camera view

    • Transformation

  • g: Translation
  • g + x/y/z: Translation along x/y/z axis
  • g + Shift + x/y/z: Translation along plane with normal axis x/y/z
  • r: Rotation
  • r + x/y/z: Rotation along x/y/z axis
  • r + Shift + x/y/z: Rotation along plane with normal axis x/y/z
  • s: Scaling
  • s + x/y/z: Scale along x/y/z axis
  • s + Shift + x/y/z: Scale along plane with normal axis x/y/z
  • Alt + g/r/s: Restore to default translation/rotation/scale
  • Shift + Mouse-right-click: Move pivot point

    • Add/Delete

  • Shift + a: Add menu
  • x/Del: Delete object
  • m: Move to collection

    • Object mode

  • Ctrl + j: Join object

    • Edit mode

  • Tab: To edit mode
  • p: Seperate
  • 1/2/3: Point/Edge/Face selection mode(Shift to add into selction mode)
  • Alt + Mouse-left-click: Loop selection
  • E: Extrude tool
  • E + Mouse-left-click: Create duplicate faces
  • Ctrl + Mouse-right-click: Extrude to cursor
  • Ctrl + r: Loop cut tool(scroll Mouse-wheel to add more loop)
  • Ctrl + b: Bevel tool
  • Ctrl + Shift + b: Bevel tool for corners
  • k: Knife tool

    • Pose mode

  • [During object mode or pose mode]Ctrl + Tab: Object mode to pose mode, or vice versa
  • Ctrl + p: Parent Armature
  • Alt + p: Clear parent
  • m: Change bone layer

    • Keyframe tools

  • Alt + Mouse-wheel-scroll: Move frame
  • Space: Play/Pause animation
  • [Mouse-cursor on certain attribute]i: Insert keyframe according to that attribute
  • [Mouse-cursor on timeline]m: Add marker to timelime
  • [Mouse-cursor on marker region]Ctrl + m: Rename marker
  • [Mouse-cursor on dope sheet]t: Set keyframe interpolation mode

    • Sculpt mode

  • Ctrl: Carve inward(Default: carve outward)/Deform along average normal direction
  • f: Adjust brush size
  • Shift + f: Adjust bruch Strength
  • c: Clay brush
  • l: Layer brush
  • i: Inflate brush
  • Shift + c: Crease brush
  • s: Smooth brush
  • Shift(hold): Apply smooth brush
  • Shift + t: Flattening brush
  • p: Pinch brush
  • g: Grab brush
  • k: Snake hook brush

    • Pie menu

  • z: Shading mode
  • Shift + s: Snap menu
